# Web Services 


#### Set up virtual environment

**Create virtual environment** 
```
python3 -m venv env
```
**Activate the virtual environment**
```
source env/bin/activate
```
**Install Python libraries**
```
pip install -r requirements.txt
```
**Run script**
```
python main.py
```
**Deactivate the virtual environment**
```
deactivate
```
